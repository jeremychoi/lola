# Code Review Skill

Provides a framework for thorough and constructive code reviews.

## Review Checklist

### Functionality
- Code works as intended
- Edge cases are handled
- Error handling is appropriate

### Code Quality
- Clear naming conventions
- No code duplication
- Appropriate abstraction level
- Comments where necessary

### Security
- No hardcoded secrets
- Input validation present
- SQL injection prevention
- XSS prevention

### Testing
- Unit tests for new code
- Integration tests where needed
- Tests are readable and maintainable

## Providing Feedback

1. Be specific and actionable
2. Explain the "why" behind suggestions
3. Distinguish between blockers and nice-to-haves
4. Acknowledge good patterns
5. Ask questions rather than make demands
